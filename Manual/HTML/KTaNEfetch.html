Keep Talking and Nobody Explodes Mod KTaNEfetch

On the Subject of KTaNEfetch

The creator of this mod uses Arch, BTW.

This module will use information from the ktanefetch.py command line program to deliver system information about the current computer. Use the ktanefetch program to determine which system information must be submitted.

How to Use ktanefetch.py

Python is required to run the script. Once downloaded, follow the instructions in this module's repo's README. You will be presented with a menu to select different system info components. Use the up and down arrow keys, or J for down and K for up, then press Space to select a component. Once finished, press Enter.

After all appropriate components have been selected, you will be given a 4-digit code. Enter this code on the module, and it will check for the proper components.

Components

Component Condition
Hostname The hostname's length is longer than the number of modules
Kernel The sum of the digits in the kernel version is greater than or equal to the sum of the digits in the serial number
Uptime The amount of uptime is less than the time at the moment of code entry multiplied by the number of batteries
Shell The shell is not the default (Defaults are CMD/Powershell on Windows, zsh on MacOS, bash on Linux) or the gameplay room is not the vanilla room
Desktop Environment See below page
CPU The module has struck before and any CPU is from AMD, or the module has not struck before and any CPU is from Intel
GPU At least one CPU and GPU are both made by AMD xor there are no modules with "Cruel" in the name
RAM The percentage of solved modules at the time of code entry is less than the percentage of RAM used at the time of code generation
