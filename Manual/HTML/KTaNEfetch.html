On the Subject of KTaNEfetch

The creator of this mod used to use Arch, BTW.

This module will gather information about system software and hardware (the OS, hostname, shell, desktop environment, CPU and GPU). Click your distribution's logo to turn the module into a terminal multiplexer, and solve each stage to complete the module. Navigate between panes with Ctrl+b, then an arrow key.

VimBeGood

The module will present you with multiple challenges, to be completed with a keyboard and Vim motions only:

  • Given a set of words such as bar bar bar far bar, jump to the unique word and delete it. Use w to jump forward one word, b to jump backwards, and d then w to delete the word.
  • With the navigation keys specified below and the x key to delete, remove the x on screen.
  • By entering a number, then j or k, jump multiple lines at once to the line that reads DELETE_ME. Then, press d twice to delete the line.

Use h to go left, j to go down, k to go up, and l to go right. Using the arrow keys during this stage will warrant a strike.
